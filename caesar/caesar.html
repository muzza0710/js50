This is Caesar

This is an implementation of cs50 problem set Caesar where a string of plain text is encrypted based on a provided key. the encryption simply shifts each letter by the key, so with a key of 1, 'a' would become 'b' etc

an important part of this task is to ensure that letters 'wrap' correctly, so in the above eg 'z' would become 'a', also letters should maintain their original case and any none alphabet characters (punctuation, numbers etc) should be left unchanged.

Finally if a key above 26 is provided this should also wrap, so a key of 27 for example would be the equivalent of a key of 1.
